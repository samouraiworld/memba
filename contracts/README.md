# Gno Realms

This directory will contain Gno realm contracts.

DAO governance frontend is available since v4.0.0 and targets the `gno.land/r/samcrew/memba_dao` realm.
DAO Factory (v5.0.0) enables deploying custom governance realms from the Memba UI via `MsgAddPackage`.

See [ROADMAP.md](../ROADMAP.md) for the timeline.
